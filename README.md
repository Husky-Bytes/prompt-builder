# Prompt Builder

A powerful and visual prompt construction tool designed to help you craft, manage, and refine complex prompts using a modular block-based approach.

## Features

-   **Modular Design**: Build prompts using reusable blocks.
-   **Visual Organization**: Color-code blocks for easy identification and grouping.
-   **Dynamic Editing**: 
    -   Drag and drop blocks to reorder.
    -   Inline text editing with auto-growing fields.
-   **Real-time Stats**: Word and character counts update instantly as you build.
-   **Library Management**:
    -   Save, edit, and organize prompts in folders.
    -   "Save As" functionality to easily create variations.
    -   Search and filter your block collection.
-   **Export/Import**: Backup your library and blocks to JSON to share or enable moving between devices.
-   **Usage Tracking**: See at a glance how many times a block is used in your current prompt.

## Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Husky-Bytes/prompt-builder.git
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the development server**:
    ```bash
    npm run dev
    ```

## Usage

1.  **Blocks**: Create blocks for common prompt segments (e.g., "Cinematic Lighting", "4K Resolution", "Cyberpunk Style"). Assign colors to categorize them.
2.  **Builder**: Drag blocks into the builder area. Mix them with custom text segments. Rearrange them to find the perfect flow.
3.  **Library**: Save your masterpiece. key prompts organized in folders.
